使い方
------
### 説明 ###
![w1](https://user-images.githubusercontent.com/58340131/75414526-2a586e80-596c-11ea-80d8-fb53bda57f57.jpg)
新規の方は始めにユーザー情報を登録してください。  

<img width="1440" alt="スクリーンショット 2020-02-27 14 33 43" src="https://user-images.githubusercontent.com/58340131/75415244-67256500-596e-11ea-9f98-7ffe00e26594.png">
ユーザには、まず自分の車の情報を登録してもらいます。  
車情報は全て埋めてください。

<img width="1440" alt="スクリーンショット 2020-02-27 14 37 35" src="https://user-images.githubusercontent.com/58340131/75415349-b9ff1c80-596e-11ea-9545-eefb1d77634c.png">
登録が済みますと、次に交換したパーツの情報を登録してもらいます。  
同じ名前は使えません。

<img width="1440" alt="スクリーンショット 2020-02-27 14 44 49" src="https://user-images.githubusercontent.com/58340131/75415714-bfa93200-596f-11ea-8ad9-9c8a4aa9cb94.png">
次にパーツの詳細に行き、交換日、次回交換推奨日、次回交換推奨距離を入力してください。

<img width="1440" alt="スクリーンショット 2020-02-27 14 46 21" src="https://user-images.githubusercontent.com/58340131/75415801-faab6580-596f-11ea-8a3b-d95e012b219a.png">
管理車、ユーザーはこの画面を見て、次の交換推奨日、次回点検日までがどれぐらいなのかわかります。

日常点検
------
日常点検では、乗車前の点検方法が記載してあります。
